# About

Front-end for [Fswap contracts](https://github.com/floriangaspard/fswap-contracts)
Contracts are deployed on Sepolia testnet only.

Built with React + TypeScript and TailwindCSS.

# Installation

```
npm install
```

# Getting started

## Start development server

```
  npm run start
```

## Build and run

1. Build

```
  npm run build
```
